# **App Name**: QuizMasterAI

## Core Features:

- AI Quiz Generation: Generate quizzes using Google Gemini based on topic, difficulty, and question count.
- Quiz Taking: Take generated quizzes with features like full screen, no right click, screen shot detection, and tab switching detection, submitting quiz at the sign of any violations.
- User Authentication: Authentication using email/password and Google OAuth.
- Admin Dashboard: Tabbed interface for quiz creation, management, and result analytics.
- Anti-Cheat System: Implement full-screen mode enforcement, screenshot prevention, context menu disable, tab switching detection, and text selection disable. LLM reasoning tool can detect potential cheating and issue appropriate warnings to users.
- Database Integration: Database for profiles, user roles, quizzes, questions, and quiz attempts with row level security policies.
- Result Analytics: Dashboard with charts and attempt history to analyze quiz results.

## Style Guidelines:

- Primary color: Vibrant cyan (#00FFFF) for a futuristic cyberpunk theme.
- Background color: Dark desaturated blue (#0A0F12) to enhance the neon effect.
- Accent color: Electric magenta (#FF00FF) to complement cyan.
- Body and headline font: 'Space Grotesk', a sans-serif font for a modern, techy feel, with 'Inter' for any longer body text that might be present.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use neon-style icons from Lucide React to match the cyberpunk theme.
- Grid background pattern using cyan lines at 3% opacity to create a cyberpunk aesthetic.
- Subtle neon glow animations for text and buttons to enhance the visual experience.
